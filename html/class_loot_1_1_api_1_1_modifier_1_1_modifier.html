Detailed Description

Defines a modifier, which is an unloaded GlobalItem Making it a GlobalItem gives easy access to all hooks The various hooks are called by our own GlobalItem In your Modifier, it is safe to assume when one of the hooks is called, that item currently is modified by this modifier
