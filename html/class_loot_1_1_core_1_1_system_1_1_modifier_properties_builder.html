Detailed Description

The ModifierPropertiesBuilder implements the builder pattern for ModifierProperties It provides a structured way to create ("build") an instance of ModifierProperties It is used in Modifier.GetModifierProperties to dynamically build the ModifierProperties for that Modifier.
